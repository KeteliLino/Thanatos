# Thanatos

---

## Introdução
O Thánatos é um sistema desenvolvido em VB.NET com foco em agilizar, organizar e centralizar processos comuns em funerárias e cemitérios, unificando tudo em uma interface moderna, intuitiva e prática.

O objetivo é reduzir erros, evitar retrabalho, aumentar a eficiência e garantir que as informações estejam sempre organizadas e acessíveis.

## Requisitos Funcionais 

| **Nº** | **Requisito Funcional** |
|--------|---------------------------|
| **RF01** | O Administrador poderá manter os serviços, velórios, jazigos, funcionários, falecidos, salas e cremações. |
| **RF02** | O Administrador poderá agendar uma sala de cremação para uma cremação em um horário ainda não alugado. |
| **RF03** | O Recepcionista poderá consultar dados sobre jazigos, velórios, falecidos e cremações. |
| **RF04** | O Administrador poderá agendar uma sala de cerimônia para um velório em um horário ainda não alugado. |
| **RF05** | O Financeiro poderá consultar, cadastrar e alterar os orçamentos. |
| **RF06** | O Financeiro poderá consultar os serviços e os falecidos. |
| **RF07** | Todos os atores poderão ter uma rápida visualização de dados importantes de velórios, orçamentos e cremações. |
